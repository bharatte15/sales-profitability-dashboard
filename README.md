# sales-profitability-dashboard
End-to-end analysis of Global Superstore operations using Python. This project processes 50,000+ records to uncover seasonal trends, regional profit distributions, and product performance. Includes data cleaning, feature engineering for margin analysis, and automated KPI visualizations to drive data-based decisions.

Overview: 
This project is an end-to-end data analytics solution designed to analyze sales performance and profitability. It focuses on identifying key business drivers, uncovering trends, and generating actionable insights to support data-driven decision-making.

Business Objectives: 
Analyze overall sales and profit performance
Identify high-performing and loss-making products
Understand the impact of discounts on profitability
Evaluate regional and category-wise performance
Provide insights to improve profit margins

Tools & Technologies:
Python (Data Cleaning & Transformation)
Microsoft Power BI (Dashboard & Visualization)
SQL (Data Analysis concepts)
Excel / CSV (Dataset handling)

Project Workflow:
Data Collection – Imported raw sales dataset
Data Cleaning – Handled missing values, formatted columns, feature engineering
Data Transformation – Created year, month, and profit margin metrics
Data Analysis – Performed exploratory analysis using Python
Visualization – Built interactive dashboards in Power BI
Insights Generation – Identified key business trends and recommendations

Dashboard Features:
KPI Cards: Total Sales, Total Profit, Profit Margin
Sales vs Profit Trend Analysis
Region-wise Performance
Top Products by Profit
Discount vs Profit Correlation
Interactive Filters (Region, Category, Year)

Key Insights:
High discounts negatively impact profitability
Certain regions generate high sales but lower profit margins
A small group of products contributes significantly to total revenue
Profitability varies across categories and customer segments

Business Recommendations:
Optimize discount strategies to improve margins
Focus on high-profit regions and products
Reduce dependency on low-margin sales
Use data insights for strategic decision-making

Future Enhancements:
Sales forecasting using time-series models
Customer segmentation for targeted strategies
Integration with real-time data sources
